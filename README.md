# PhotoSorter

**PhotoSorter** — это простая и быстрая программа для автоматического наведения порядка в ваших семейных фото- и видеоархивах. Больше никаких тысяч файлов в одной куче — программа сама разложит всё по аккуратным папкам.

Вам **не нужно** устанавливать Python или уметь программировать. Просто скачайте версию для вашей операционной системы по ссылке ниже:

*   **Для macOS:** Скачайте файл `PhotoSorter.dmg` (или `.app`), перетащите его в папку «Программы» и запустите.

## Что умеет программа?

*   **Сортировка по дате съёмки:** программа «умная» - она смотрит не на то, когда файл был скопирован на компьютер, а на реальную дату съёмки из метаданных (EXIF) самой фотографии.
*   **Удобная глубина папок:** вы сами выбираете, как разложить файлы — только по годам (*2025/*), по годам и месяцам (*2025/07_Июль/*) или с точностью до дня.
*   **Разделение фото и видео:** можно включить настройку, чтобы видеоролики автоматически складывались в отдельные подпапки и не перемешивались с фотографиями.
*   **Очистка от скриншотов:** программа умеет распознавать скриншоты и бережно изолирует их в отдельную папку `Screenshots`, чтобы они не засоряли ваши главные воспоминания.
*   **Интерактивный предпросмотр:** перед тем как физически переместить файлы, программа откроет просторное окно и покажет будущую структуру. Вы сможете наглядно изучить список и убрать лишнее крестиком (`✕`).
*   **Быстрое превью картинок:** просто наведите мышку на имя файла в списке, и рядом мгновенно появится миниатюра фотографии. 
*   **Безопасный режим:** боитесь что-то потерять? Включите режим копирования, и программа не тронет исходные файлы, а просто создаст их упорядоченные копии в новом месте.
*   **Автообновление:** программа сама проверит наличие новой версии и предложит обновиться.

## Как пользоваться?

1. В верхнем поле выберите папку, где сейчас царит хаос (например, скинутые файлы с телефона).
2. Во втором поле выберите или создайте пустую папку, куда хотите сложить результат.
3. Выставите галочки под свои задачи и нажмите **Открыть предпросмотр**.
4. Проверьте результат в открывшемся окне, наводя мышку на файлы. Если всё нравится — возвращайтесь в главное окно и смело жмите **Запустить сортировку**.

<summary>Инструкция для разработчиков (техническая информация)</summary>

### Технологический стек
* Python 3.12+
* CustomTkinter (GUI)
* Pillow (Генерация миниатюр)

### Запуск из исходного кода
1. Клонировать репозиторий: `git clone https://github.com/almiraira/photosorter.git`
2. Создать окружение: `python3 -m venv .venv && source .venv/bin/activate`
3. Установить зависимости: `pip install -r requirements.txt`
4. Запустить: `python run_gui.py`

### Сборка через PyInstaller
* **Windows:** `pyinstaller --noconsole --onefile --icon=icon.ico run_gui.py`
* **macOS:** `pyinstaller --noconsole --onefile --name="PhotoSorter" --icon=icon.icns --add-data ".venv/lib/python3.12/site-packages/customtkinter:customtkinter" run_gui.py`

> **Важное примечание для пользователей macOS:**
> Так как приложение распространяется как open-source без платной подписи Apple Developer, при первом запуске macOS может заблокировать программу с предупреждением о «неизвестном разработчике». 
> 
> **Как запустить:**
> 1. Перетащите `PhotoSorter` в папку **«Программы»** (Applications).
> 2. Нажмите на иконку приложения **правой кнопкой мыши (или двумя пальцами по тачпаду)** и выберите **Открыть** (Open).
> 3. В появившемся окне появится кнопка **«Открыть»** — нажмите её. Это нужно сделать всего один раз, дальше программа будет запускаться как обычно.
> 4. *Если кнопка не появилась:* зайдите в *Системные настройки ➔ Конфиденциальность и безопасность*, прокрутите вниз и нажмите кнопку **«Подтвердить вход»** (Open Anyway) напротив PhotoSorter.
